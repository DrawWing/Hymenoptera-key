# Hymenoptera of Europe: taxonomic key

The taxonomic key in HTML format.

This key is based on the work of Gauld and Bolton (1988).
Gauld, I., Bolton, B. (1988). The Hymenoptera. British Museum (Natural History), London.

The key was prepared using DKey software (Tofilski 2018).
Tofilski A., 2018. DKey software for editing and browsing dichotomous keys. ZooKeys 735: 131-140. https://doi.org/10.3897/zookeys.735.21412
